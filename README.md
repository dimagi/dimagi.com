# dimagi.com

## Getting Started

1. Install `virtualenv` for `Python 3.6` and `pip3` and make your virtual environment.
2. Install `pipenv`
 ```
 pip3 install pipenv
 ```
 
3. Install from `Pipfile`
 ```
 pipenv install
 ```

4. Install the [Heroku CLI](https://devcenter.heroku.com/articles/heroku-cli)
