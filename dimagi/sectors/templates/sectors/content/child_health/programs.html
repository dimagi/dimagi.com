{% blocktrans %} Child Health Programs {% endblocktrans %}

{% trans "Problem" %}: {% blocktrans %} Child health programs require extensive health worker trainings. It is difficult to share child health data among stakeholders. {% endblocktrans %}

  • {% blocktrans %} Mobile applications can supplement health worker trainings with multimedia-enabled training modules {% endblocktrans %}
  • {% blocktrans %} Programs can remotely update IMCI and CCM applications, enabling smoother programmatic adjustments {% endblocktrans %}
  • {% blocktrans %} Supervisors can detect data irregularities and oversee health workers’ activities, including frequency of visits, speed of follow up, number of cases identified, length of counseling sessions, and health workers that aren’t adhering to IMCI and CCM protocols {% endblocktrans %}
  • {% blocktrans %} Program managers can conduct health surveillance, monitoring, and evaluation of IMCI/ CCM programs, activating timely response where IMCI implementation is not successful {% endblocktrans %}
  • {% blocktrans %} Ministries of health can access easy-to-read reports on child health data by geographic area and make the appropriate programmatic decisions {% endblocktrans %}
