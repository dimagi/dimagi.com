{% blocktrans %} Diagnostics and Lab Tracking {% endblocktrans %}

{% trans "Problem" %}: {% blocktrans %} During outbreaks, labs are established hastily and results to patients are delayed. {% endblocktrans %}

  • {% blocktrans %} CommCare data entered at the lab site will be automatically uploaded as soon as a connection is available {% endblocktrans %}
  • {% blocktrans %} CommCare enables different authorization levels, for personnel to access data on a web-based platform to consult results in real time and to upload new test results {% endblocktrans %}
  • {% blocktrans %} CommCare supports user-configurable reports that can be exported via CSV or to third party analytics software such as Tableau via APIs {% endblocktrans %}
  • {% blocktrans %} Test results can be automatically sent via SMS from the central database to recipients {% endblocktrans %}
  • {% blocktrans %} CommCare is compatible with QR code readers, and can be configured to receive data from laboratory and point-of-care devices with digital outputs via APIs {% endblocktrans %}
