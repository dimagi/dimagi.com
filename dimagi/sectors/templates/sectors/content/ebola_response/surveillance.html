{% blocktrans %} Surveillance and Contact Tracing {% endblocktrans %}

{% trans "Problem" %}: {% blocktrans %} Controlling outbreaks requires identifying suspected cases before they can expose other people. {% endblocktrans %}

  • {% blocktrans %} Frontline workers can use CommCare to identify contacts and follow up with them over time {% endblocktrans %}
  • {% blocktrans %} Patients can be easily tracked across multiple visits thanks to registration modules {% endblocktrans %}
  • {% blocktrans %} In-app background validations and calculations automatically notify frontline workers when a contact needs to be followed up with, or can be declared safe after 21 days {% endblocktrans %}

  • {% blocktrans %} Patient records can be automatically sent to health facilities for up to date incidence reports {% endblocktrans %}
