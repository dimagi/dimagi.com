{% blocktrans %} TB Programs and Clinics {% endblocktrans %}

{% trans "Problem" %}: {% blocktrans %} TB programs and clinics have limited trained personnel to register and follow up with TB patients and provide treatment. {% endblocktrans %}

  • {% blocktrans %} Collect patient information to create real-time TB and MDR-TB profiles of specific areas {% endblocktrans %}
  • {% blocktrans %} Integrated system allows lab results to be easily shared between clinics in different communities {% endblocktrans %}
  • {% blocktrans %} Automatically alerts district health center of new cases and others that miss subsequent visits {% endblocktrans %}
  • {% blocktrans %} Basic web reporting detailing system utilization facilitates data management {% endblocktrans %}
  • {% blocktrans %} Automatic patient appointment reminders can be configured to reduce staff time contacting patients {% endblocktrans %}
  • {% blocktrans %} GPS integration creates data for a map visualization of MDR-TB spatial spread and progression {% endblocktrans %}
  • {% blocktrans %} Interactive voice response (IVR) triages cases, reducing burden on clinic staff {% endblocktrans %}
