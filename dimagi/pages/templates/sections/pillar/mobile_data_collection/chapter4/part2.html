{% blocktrans %} How Do Your Data Requirements Affect Your Mobile Data Collection Tool? {% endblocktrans %}

{% blocktrans %} Mobile data collection tools have immense potential but there isn't a single solution that will work for everyone. As you might expect, different tools offer different features. {% endblocktrans %}

{% blocktrans %} Dimagi's User Engagement Manager Marshall Daly recommends evaluating your data needs before you start looking at different market offerings. {% endblocktrans %}

{% blocktrans %} "Like any other data collection program, start with the data you want, and then find a tool that allows you to collect that data," Daly said. {% endblocktrans %}

{% blocktrans %} Even after identifying your data requirements, selecting the right platform is often a challenge, and there are several points you should consider throughout the selection process: {% endblocktrans %}

{% include "svg/pillar/mdc/chapter4/icon5.html" %}
{# .column #}

{% blocktrans %} What format do you need to present your data in? {% endblocktrans %} {% blocktrans %} Certain data formats (e.g. audio, video, GPS coordinates, etc.) require certain features in order to capture them. {% endblocktrans %}

{# .column #}
{# .grid #}
{% include "svg/pillar/mdc/chapter4/icon6.html" %}
{# .column #}

{% blocktrans %} Will you need to update your data? {% endblocktrans %} {% blocktrans %} If your program requires that workers revisit data sources (beneficiaries, locations, etc.) to collect updated information, your platform should allow you to link and review multiple data entries with a feature such as case management. {% endblocktrans %}

{# .column #}
{# .grid #}
{% include "svg/pillar/mdc/chapter4/icon7.html" %}
{# .column #}

{% blocktrans %} Who has access to your data? {% endblocktrans %} {% blocktrans %} Some mobile data collection tools allow for multiple predefined users to view and update data from a single case, enhancing efficiency and security. {% endblocktrans %}

{# .column #}
{# .grid #}
{% include "svg/pillar/mdc/chapter4/icon8.html" %}
{# .column #}

{% blocktrans %} How do you need to share your data? {% endblocktrans %} {% blocktrans %} Think about whether you need a simple export to Excel, more complex data visualization features, or even full platform integrations with specialized reporting software. {% endblocktrans %}

{# .column #}
{# .grid #}

{% blocktrans %} Not only does the type of data you collect affect the tools and features you need to collect it, but its characteristics and those of its environment will also be important considerations when deciding on the right mobile data collection solution for your team. {% endblocktrans %}
