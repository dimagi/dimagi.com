{% blocktrans %} Your app should not only function on the device — it should thrive. {% endblocktrans %}

{% blocktrans %} Problems like slowness and app crashes should already be addressed during design and testing. However, while smaller projects might be able to design for a specific model, apps for larger scale may need to work for a wider range of device specifications. {% endblocktrans %}
