{% blocktrans %} Training is the most important part of implementing a data collection tool. This is where workers get to know how their work will change, what new skills and behaviors they need to develop, and which old approaches will no longer be helpful. {% endblocktrans %}

{% blocktrans %} This is also where they are introduced to the data collection tool. {% endblocktrans %}
