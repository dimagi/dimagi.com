{% blocktrans %} Storage and Security {% endblocktrans %}

{% blocktrans %} Certain sectors lean more heavily on this consideration than others. For some beneficiary populations and projects, such as those working with HIV patient data, privacy concerns may be much more important than others. {% blocktrans %} The actual considerations will depend on the sector you are working in. {% blocktrans %} As you consider the storage and privacy of your data, ask yourself the following: {% endblocktrans %}

  • {% blocktrans %} Does the dataset need to be de-identified before exporting and sharing? {% endblocktrans %}
  • {% blocktrans %} Do I need to protect certain data after it is entered? {% endblocktrans %}
  • {% blocktrans %} Who can have access to the data? {% endblocktrans %}
  • {% blocktrans %} How long can the data be stored? {% endblocktrans %}
